## About

These sessions started as internal knowledge-sharing within 
[Aiden](https://aiden.eu) (a Mendix consultancy) and are made 
available here for a wider audience of developers, designers, 
and the Mendix community.

Topics cover:
- UX principles that hold up across tools and time
- Design systems and visual language
- Mendix-specific implementation patterns
- The craft of low-code development

## Style

8-bit pixel illustrations on dark backgrounds, monospace typography, 
no-nonsense formatting. Built for legibility on projectors, laptops, 
and phones alike.

## Built with

- HTML / CSS / JS — static, no build step required
- GitHub Pages — free hosting, custom domain
- Lots of trial and error
